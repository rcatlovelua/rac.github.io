**Rcat's Anti-Cheat (RAC)** is a server-side anti-cheat system for **Roblox Studio**, designed to detect and prevent common exploiting and cheating behaviors in Roblox games.

RAC focuses on **security**, **performance**, and **easy integration**, while remaining flexible and extensible for developers.

---

## Features

 Detection of:
  - Speed Hacks
  - Fly (SOON) / NoClip
  - Teleport exploits
  - Anti building tools
  - Server-side checks (never trust the client)
  - Player behavior monitoring
  - Fully configurable system
  - Debug logging & detection reports
  - Modular architecture (easy to extend)
  - Lightweight & performance-friendly
